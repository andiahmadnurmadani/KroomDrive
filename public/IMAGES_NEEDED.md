# Images needed for README

Place these files in this `/public` folder:

| File | Size | Description |
|---|---|---|
| `banner.png` | 1200×400px | KroomDrive banner/header image |
| `screenshot-main.png` | 1200×700px | Main dashboard screenshot |
| `screenshot-admin.png` | 800×500px | Admin console screenshot |
| `screenshot-git.png` | 800×500px | Git panel screenshot |
| `kroombox-logo.png` | any, ~200px wide | KroomBox logo |

Tips:
- Take screenshots at 1.5x zoom for sharper images
- Use a dark browser theme for best contrast
- Crop to remove browser chrome (just the app UI)
- Compress with https://squoosh.app before uploading
